This tool allows you to scan any Ethereum wallet address for NFTs.
It will export a CSV file with a list in the format ready for Safe's CSV Airdrop app, you just need to assign the receiver and how many tokens to send, which is always 1 for erc-721 tokens.
Additionally, it will also give you the option to save a reference file for each address so you know which token contract is which by name and how many total tokens of each contract are held.

  1. Enter the wallet address/es (one per line for multiple) in the first field.
  2. Hit the "Export List of NFTs" button and wait.
  3. Download your CSV files.
